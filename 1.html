HTML 편집기로는 ATOM을 사용

> https://atom.io
> atom 설치
> 바탕화면-web이라는 이름으로 새 폴더 만들기
> atom 실행
> File-Open folder-web 선택

> New file - 1(파일명).html(확장자)
> 웹 브라우저에서 ctrl+O - web-1.html 클릭
> 하얀색 화면이 뜬다!
> 편집 화면에서 hello web을 입력 후 저장
> 웹 브라우저 새로고침
